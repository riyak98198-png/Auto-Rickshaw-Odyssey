# Bombay Ferrari – Mumbai Traffic Ecosystem

## Overview
Bombay Ferrari – The Living Mumbai Traffic Ecosystem is a mobile-friendly 3D simulation where players navigate a yellow-and-black autorickshaw through the chaotic streets of Mumbai. Every ride is unique, and the city environment dynamically changes to create a living and unpredictable traffic ecosystem.

## Gameplay
- Navigate through Mumbai streets in a **yellow-and-black autorickshaw**.
- Avoid traffic, street dogs, construction barricades, and other obstacles.
- Collect rewards such as:
  - 🍵 Chai glasses for speed boosts
  - Loose coins
  - Snacks
- Monitor **Distance**, **Speed**, and **Score** via the HUD.
- Dynamic traffic zones are indicated with labels such as **Calm Zone**.
- Encounter changing weather, including rain and puddles.

## Start Screen
- Title: **AUTORICKSHAW ODYSSEY**
- Subtitle: "Navigate the living, breathing chaos of Mumbai streets. Every ride is unique."
- Emoji: 🛺
- Tips displayed on grid:
  1. 🌧️ Rain slows you down — watch out!
  2. 🍵 Collect chai for speed boost
  3. 🐕 Dogs & vendors block the road
  4. 💰 Dodge traffic, earn coins
- Start button: **START RIDE**

## Controls
- On-screen buttons:
  - ◀ Left
  - ▶ Right
  - 🛑 Brake
  - ⚡ Accelerate
  - 📯 Horn

## HUD (Heads-Up Display)
- **Distance** – meters
- **Speedometer** – KM/H
- **Score**
- Zone label: indicates traffic condition (example: Calm Zone)
- Weather badge (e.g., ☀️ Clear Skies)
- Event banner for notifications
- Reward popup for collectibles
- Multiplier badge (e.g., ×2)

## Game Over
- Displayed on crash:
  - Emoji: 🛺💥
  - Title: **CRASHED!**
  - Score line showing Score and Distance
  - Restart button: **RIDE AGAIN**

## Technical Notes
- Built using **Three.js** for 3D rendering.
- Mobile-friendly canvas with touch controls.
- Dynamic, procedural road and traffic system.
- Smooth animations for the autorickshaw, wheels, and environment.
- Realistic 3D materials for:
  - Rickshaw body, roof, wheels, windshield
  - Vehicles: cars, buses, bikes
  - Environmental props: trees, buildings, puddles, vendors
- Dynamic lighting, fog, and shadows.
- Rain overlay with smooth transitions.

## Deployment
- This project is a single-page HTML + JS and CSS file.
- To run, either use this link : "https://riyak98198-png.github.io/Auto-Rickshaw-Odyssey/"  or open `index.html` in a browser (mobile or desktop).
- No server-side setup required.
